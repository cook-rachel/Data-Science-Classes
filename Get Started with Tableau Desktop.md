# Part 1: Introduction to Tableau Desktop
---
Are you a new user of Tableau Desktop, eager to start building visualizations? 

In this module, you'll do that and more. You'll get a quick grasp of the fundamentals, learning the key phases of the data workflow in Tableau Desktop: connecting, analyzing, and sharing. Then you'll focus on each phase and complete it yourself: connecting to data, analyzing it using essential charts, and creating an interactive dashboard for sharing your views. 

Read on to start your Tableau Desktop journey.

**Objectives**
At the end of this module, you will be able to:
- Describe the iterative Tableau Desktop workflow of connecting to, analyzing, and sharing your data.
- Connect to and preview data in Tableau Desktop.
- Describe the key areas of the Tableau Desktop workspace.
- Build a bar chart and a crosstab in the workspace.
- Use visualizations to answer questions about data.
- Bring views together in an interactive dashboard.
- Describe options for sharing your analysis.
